Play and Akka Clusters

This is a small play application that is connected with an akka cluster. Make sure you started the cluster with 

            
                sbt "project backend" "run 2551"
                sbt "project backend" "run 2552"

Cluster Dashboard

The dashboard visualize the current cluster status. Shows the state of different nodes and other metrics. It's updated via a websocket, so just fire up another cluster instance with 

sbt "project backend" run

Factorial Calculation

You can start a factorial calculation for x times on the cluster. You will see how many of the jobs are already finished and how long everything took.
